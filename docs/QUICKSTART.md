# 快速開始指南

5 分鐘內讓 Gasker Content Refresher 運作起來!

## 🚀 快速安裝 (3 步驟)

### 步驟 1: 安裝 AI Engine
```
WordPress 後台 → 插件 → 安裝插件
搜尋 "AI Engine" → 安裝 → 啟用
```

### 步驟 2: 設定 API Key
```
Meow Apps → AI Engine → Settings
選擇 OpenAI 或 Google 標籤
輸入 API Key → Save
```

### 步驟 3: 安裝本插件
```
上傳 gasker-content-refresher.zip
啟用插件
```

## ⚙️ 快速設定 (推薦配置)

前往 **設定 → Content Refresher**

### 基本設定
```
每日處理篇數: 5
冷卻天數: 30
最大文章長度: 5000
```

### AI 設定
```
AI 模型: GPT-4o
目標語言: 繁體中文
```

### 發布模式
```
✓ 待審閱 (設為待審狀態) - 推薦
```

點擊 **儲存設定**

## 🧪 測試運行

1. 點擊 **「立即執行一次」** 按鈕
2. 等待 30-60 秒
3. 查看執行結果
4. 前往 **文章 → 所有文章**
5. 檢查「待審閱」的文章

## ✅ 確認成功

檢查清單:

## 🔧 常見問題速查

### Q: 點擊「立即執行」沒反應?
A: 檢查瀏覽器控制台是否有 JavaScript 錯誤

### Q: 顯示「AI Engine 未啟用」?
A: 確認已安裝並啟用 AI Engine 插件

### Q: 執行失敗顯示 API 錯誤?
A: 檢查 API Key 是否正確,帳號是否有餘額

### Q: 文章沒有被處理?
A:
- 確認有符合條件的文章 (30 天前未更新)
- 檢查是否被排除規則過濾
- 查看執行日誌中的錯誤訊息
